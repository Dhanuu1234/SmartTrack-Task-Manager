# 🚀 SmartTrack Task Manager

A professional Flask-based Task Management System featuring Role-Based Access Control (RBAC) and seamless SQL integration.

## ✨ Features
* **User Authentication:** Secure login and registration.
* **Role-Based Access:** Different permissions for Admins and standard Users.
* **Task Management:** Create, update, and track status of tasks.
* **SQL Database:** Structured data storage for reliability.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Database:** SQL
* **Frontend:** HTML5, CSS3

## ⚙️ Installation
1. Clone the repository:
   `git clone https://github.com/Dhanuu1234/SmartTrack-Task-Manager.git`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the application:
   `python app.py`
