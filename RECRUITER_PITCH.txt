WHAT IS SMARTTRACK?
------------------------------------------------------------
SmartTrack is a full-stack web application that allows
managers to assign tasks to employees and track completion
in real-time — exactly like Jira, Asana, and internal tools
used at TCS, Infosys, Wipro, and Accenture every day.

WHAT TO SAY IN AN INTERVIEW
------------------------------------------------------------
Q: "Tell me about a project you've built."

A: "I built SmartTrack, a full-stack task management
system in Python Flask and SQL. It has role-based login
so admins can assign tasks to employees and track status,
while employees can only see and update their own tasks.
I used SQLite with SQL Views and subqueries for the
reporting dashboard. The app is live at [URL] — I can
show you the code on GitHub if you'd like."

Q: "Why did you choose Flask over Django?"

A: "Flask gives more control over the architecture for
a project this size. I wanted to understand every line
of the routing, authentication, and database logic
myself rather than using Django's built-in abstractions.
This helped me learn REST principles more deeply."

Q: "How would you scale this for production?"

A: "I'd migrate from SQLite to PostgreSQL or MySQL,
add JWT-based REST APIs instead of session auth,
containerize with Docker, and deploy on AWS EC2
with a CI/CD pipeline using GitHub Actions."
